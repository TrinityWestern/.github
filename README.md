# TWU GitHub Organization Repository

This repository is intended to be published as `trinitywestern/.github`.

GitHub treats a `.github` repository as a special organization-level repository. It can be used to store shared community health files and the public organization profile shown on the GitHub organization page.

## Purpose

Use this repository to manage content that applies across the `trinitywestern` GitHub organization, including:

- the organization profile README
- contribution guidance
- issue and pull request templates
- other shared GitHub metadata

## Current Contents

- [`profile/README.md`](./profile/README.md): organization profile content displayed on the GitHub organization page
- [`CONTRIBUTING.md`](./CONTRIBUTING.md): contribution guidance for this repository
- [`new-repo-checklist.md`](./new-repo-checklist.md): checklist for creating new repositories
- [`repo-naming.md`](./repo-naming.md): repository naming guidance

## GitHub Behavior

When this repository is named `.github` under the `trinitywestern` organization:

- `profile/README.md` is used for the organization profile at `github.com/trinitywestern`
- supported community health files can be shared across repositories in the organization

## Maintenance

Keep this repository focused on organization-wide GitHub standards and publicly shared documentation. Avoid adding private documentation, project-specific code or application assets here.
